1เข้าสู่ระบบ
Role: ทุกผู้ใช้งาน
ผู้ใช้งาน Login เพื่อเข้าใช้งานระบบตามสิทธิ์ของตนเอง
ผลลัพธ์: เข้าสู่หน้าหลักของระบบ
2ดูภาพรวมสต็อก
Role: ผู้จัดการ / Admin
ดูจำนวนสินค้าทั้งหมด รับเข้า เบิกจ่าย และสินค้าใกล้หมดจาก Dashboard
ผลลัพธ์: เห็นสถานะคลังแบบรวดเร็ว
3ตรวจสินค้าในคลัง
Role: พนักงานทั่วไป / Admin
ตรวจรายการสินค้า จำนวนคงเหลือ หน่วยนับ และสถานะต่ำกว่ากำหนด
ผลลัพธ์: รู้ว่าสินค้าใดต้องเติมหรือควบคุม
4รับเข้า / สร้างใบเบิก
Role: พนักงานทั่วไป
ทีมคลังบันทึกรับเข้า ส่วนผู้ขอเบิกสร้างใบเบิกพร้อมระบุรายการและจำนวน
ผลลัพธ์: เกิดรายการเคลื่อนไหวหรือใบเบิกรออนุมัติ
5อนุมัติใบเบิก
Role: ผู้จัดการ
ตรวจสอบใบเบิก รายการสินค้า และจำนวนที่ขอ ก่อนอนุมัติหรือปฏิเสธ
ผลลัพธ์: ใบเบิกผ่านการควบคุมก่อนตัดสต็อก
6ตัดสต็อกและติดตาม
Role: Admin / ผู้จัดการ
หลังอนุมัติ ระบบปรับยอดคงเหลือ และเก็บประวัติรับเข้า-เบิกจ่ายย้อนหลัง
ผลลัพธ์: ตรวจสอบย้อนหลังและทำรายงานได้
พนักงานทั่วไปตรวจสอบสินค้า สร้างใบเบิก และติดตามสถานะคำขอของตนเอง
ผู้จัดการดูภาพรวมสต็อก ตรวจสอบข้อมูล และอนุมัติใบเบิกก่อนตัดสต็อก
Adminดูแลข้อมูลพื้นฐาน ผู้ใช้งาน สิทธิ์ รายการสินค้า และความถูกต้องของระบบ
